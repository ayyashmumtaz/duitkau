Petunjuk Setup MySQL
Docker:
docker run -d --name duitkau-mysql \
  -e MYSQL_ROOT_PASSWORD=root \
  -e MYSQL_DATABASE=duitkau \
  -e MYSQL_USER=duitkau \
  -e MYSQL_PASSWORD=duitkau123 \
  -p 3306:3306 mysql:8.4 Install driver:
npm install mysql2 && npm uninstall sqlite3
Setelah dikonfigurasi, schema database akan dibuat otomatis dan akun default di-seed.
